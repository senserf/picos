PIcos Software Development Assistant
====================================


config.prj:

P(CO,xx), where xx is:

MB - multiple boards (0 or 1)
BO - board or board list, depending on MB; in the latter case, the list is
     { { suf1 BOARD } { suf2 BOARD } ... }
