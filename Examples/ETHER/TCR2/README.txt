This is the same protocol as in TCR1 (i.e., Tree collision resulution by
Capetanakis) with the transmitter reorganized as to make it
usable in the implementation of CSMA/CD-DP (directory DP).
The common files included by the files in this directory can be found
in 'IncLib'.
To create a simulator instance, just execute 'mks' in this directory.
