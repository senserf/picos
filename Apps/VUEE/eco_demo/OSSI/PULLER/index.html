Usage

Each of the "From/Upto" fields should include a date/time in several popular formats, e.g., Apr 29, 2009 12:33, 17:15:03 (refers to the current day), 09-04-29 14:44, and so on. Both fields are mandatory.

The "Rig Id" field should (must) include the EMS device number (415, 463).

The "Collector" field may be empty (all collectors) or may include a single collector number. If you put the word "events" into it, you will see event messages (i.e., all received messages that are not samples).
